# Trabajo Práctico Integrador - Organización Empresarial

## Automatización del Proceso de Solicitud de Vacaciones mediante Chatbot

### Alumno
Ignacio Merlo

### Descripción

Este proyecto propone la automatización del proceso de solicitud de vacaciones mediante un chatbot capaz de validar legajos, consultar saldo disponible y registrar solicitudes de manera automática.

### Objetivos

- Reducir tiempos de gestión.
- Disminuir errores administrativos.
- Mejorar la experiencia de los empleados.
- Aplicar conceptos de BPMN 2.0 y Organización Empresarial.

### Estructura

- docs: documentación del proyecto.
- diagramas: diagramas BPMN AS-IS y TO-BE.
- evidencias: capturas y evidencias de desarrollo.

### Herramientas utilizadas

- BPMN 2.0
- Draw.io
- GitHub
- ChatGPT
- Python
